## Fudan Analyzer

### 分词器说明

该分词软件来源于复旦大学自然语言处理实验室，[GitHub](https://github.com/xpqiu/fnlp/)项目已经上线。

### 分词器特性

1. 信息检索： 文本分类 新闻聚类
2. 中文处理： 中文分词 词性标注 实体名识别 关键词抽取 依存句法分析 时间短语识别
3. 结构化学习： 在线学习 层次分类 聚类 精确推理
